HYPER TEXT MARKUP LANGUAGE SEMPLICE

MANUALE DI UTILIZZO DEL LINGUAGGIO HTML

HTML (HyperText Markup Language) è il linguaggio di markup fondamentale per creare pagine web.

Questo manuale ti guiderà attraverso le basi dell’HTML, fornendo informazioni su come strutturare i tuoi documenti, utilizzare i tag e creare contenuti interattivi.

1. Struttura di Base di un Documento HTML

Ogni documento HTML inizia con una dichiarazione che indica al browser che tipo di documento sta per visualizzare. Ecco la struttura base: 

<!DOCTYPE html>
<html lang="it">
 <head>
   <meta charset="UTF-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Titolo della Pagina/title>
 </head>
 <body>
   <h1>Ciao, Mondo!/h1>
   <p>Benvenuti nel mondo di HTML.</p>
 </body>
</html>

Spiegazione dei Componenti:

  • <!DOCTYPE html>: Dichiara che si tratta di un documento HTML5.
  • <html>: Inizio del documento HTML.
  • <head>: Contiene metadati, link a fogli di stile e il titolo della pagina.
  • <body>: Contiene il contenuto visibile della pagina.

2. I Tag HTML Fondamentali

2.1. Intestazioni

Le intestazioni sono utilizzate per creare titoli di vari livelli, tag h1,h2,...,h6: 

<h1>Intestazione di Livello 1</h1>
<h3>Intestazione di Livello 2</h3>
<h3>Intestazione di Livello 3</h3>

2.2. Paragrafi

Per creare un nuovo paragrafo, usa il tag p: 

<p>Questo è un paragrafo.</p>
<p>Questo è un altro paragrafo.</p>

Questo è un paragrafo.

Questo è un altro paragrafo.

2.3. Link

Per inserire un link, utilizza il tag a: 

<a href="https://www.example.com">Visita Example.com</a>
Visita Example.com

2.4. Immagini

Per inserire un'immagine, utilizza il tag img: 

<img src="immagine.jpg" alt="Descrizione dell'immagine">

    <img src="https://ssl.gstatic.com/ui/v1/icons/mail/rfr/logo_gmail_lockup_dark_1x_r5.png" alt="GMail logo">

2.5. Liste

Puoi creare liste ordinate o non ordinate: 

<ul>
   <li>Elemento 1</li>
   <li>Elemento 2</li>
</ul>

<ol>
   <li>Primo Elemento</li>
   <li>Secondo Elemento</li>
</ol>
  • Elemento 1
  • Elemento 2
  1. Primo Elemento
  2. Secondo Elemento

3. Formattazione del Testo

Puoi formattare il testo usando vari tag: 

<strong>Testo in grassetto</strong>
<em>Testo in corsivo</em>
<small>Testo in piccolo</small>
Testo in grassetto
Testo in corsivo
Testo in piccolo

4. Moduli

I moduli consentono di raccogliere informazioni dagli utenti, tag form: 

<form action="/invia" method="post">
<label for="nome">Nome:</label>
<input type="text" id="nome" name="nome" required>
<input type="submit" value="Invia">
</form>

5. Tabelle

Puoi creare tabelle con il tag table: 

<table>
<tr>
<th>Intestazione 1</th>
<th>Intestazione 2</th>
</tr>
<tr>
<td>Dati 1</td>
<td>Dati 2</td>
</tr>
</table>
Intestazione 1 Intestazione 2
Dati 1 Dati 2
Dati 1 Dati 2

6. Incorporamento di Video e Audio

Per aggiungere contenuti multimediali:, tag video, audio controls: 

<video controls>
<source src="video.mp4" type="video/mp4">
    Il tuo browser non supporta il video.
</video>

<audio controls>
<source src="audio.mp3" type="audio/mp3">
  Il tuo browser non supporta l'audio.
</audio>

7. Best Practices

7.1 Tag Semantici

Uso dei tag semantici: Preferisci tag come header, footer, article, section per migliorare la leggibilità e l'accessibilità.

7.2 Commenti

Commenti nel codice: Usa i commenti per annotare il tuo codice: 

<!-- Questo è un commento a linea singola-->

<!--
   ********************************************* 
   *** Questo è un commento
   *** multi-linea
   *** ...
   *********************************************
-->


<!-- (commento codice html) -----------------
   <h3>testo non visualizzato</h3>
   <p>bla, bla, bla ...</p>
-->

7.3 Validazione

Utilizza strumenti di validazione per assicurarti che il tuo codice sia conforme agli standard HTML.
