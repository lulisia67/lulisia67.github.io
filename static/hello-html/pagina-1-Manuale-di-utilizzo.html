HYPER TEXT MARKUP LANGUAGE SEMPLICE

-MANUALE DI UTILIZZO DEL LINGUAGGIO HTML-

HTML (HyperText Markup Language) è il linguaggio di markup fondamentale per creare pagine web. Questo manuale ti guiderà attraverso le basi dell’HTML, fornendo informazioni su come strutturare i tuoi documenti, utilizzare i tag e creare contenuti interattivi. 1. Struttura di Base di un Documento HTML
Ogni documento HTML inizia con una dichiarazione che indica al browser che tipo di documento sta per visualizzare. Ecco la struttura base:

html
Copia codice
!DOCTYPE html>
html lang="it">
head>
meta charset="UTF-8">
meta name="viewport" content="width=device-width, initial-scale=1.0">
title>Titolo della Pagina/title>
/head>
body>
h1>Ciao, Mondo!/h1> p>Be
nvenuti nel mondo di HTML./p> /body>

/html>
Spiegazione dei Componenti:
!DOCTYPE html>: Dichiara che si tratta di un documento HTML5.
html>: Inizio del documento HTML.
head>: Contiene metadati, link a fogli di stile e il titolo della pagina.
body>: Contiene il contenuto visibile della pagina.
2. I Tag HTML Fondamentali
2.1. Intestazioni
Le intestazioni sono utilizzate per creare titoli di vari livelli:
html
Copia codice
h1>Intestazione di Livello 1/h1>
h2>Intestazione di Livello 2/h2>
h3>Intestazione di Livello 3/h3>
2.2. Paragrafi
Per creare un nuovo paragrafo, usa il tag p>:
html
Copia codice
p>Questo è un paragrafo./p>
2.3. Link
Per inserire un link, utilizza il tag a>:
html
Copia codice
a href="https://www.example.com">Visita Example.com/a>
2.4. Immagini
Per inserire un'immagine, utilizza il tag img>:
html
Copia codice
img src="immagine.jpg" alt="Descrizione dell'immagine">
2.5. Liste
Puoi creare liste ordinate o non ordinate:
html
Copia codice
ul>
li>Elemento 1/li>
li>Elemento 2/li>
/ul>
ol>
li>Primo Elemento/li>
li>Secondo Elemento/li>
/ol>
3. Formattazione del Testo
Puoi formattare il testo usando vari tag:
html
Copia codice
strong>Testo in grassetto/strong>
em>Testo in corsivo/em>
small>Testo in piccolo/small>
4. Moduli
I moduli consentono di raccogliere informazioni dagli utenti:
html
Copia codice
form action="/invia" method="post">
label for="nome">Nome:/label>
input type="text" id="nome" name="nome" required>
input type="submit" value="Invia">
/form>
5. Tabelle
Puoi creare tabelle con il tag table>:
html
Copia codice
table>
tr>
th>Intestazione 1/th>
th>Intestazione 2/th>
/tr>
tr>
td>Dati 1/td>
td>Dati 2/td>
/tr>
/table>
6. Incorporamento di Video e Audio
Per aggiungere contenuti multimediali:
html
Copia codice
video controls>
source src="video.mp4" type="video/mp4">
Il tuo browser non supporta il video.
/video>
audio controls>
source src="audio.mp3" type="audio/mp3">
Il tuo browser non supporta l'audio.
/audio>
7. Best Practices
Uso dei tag semantici: Preferisci tag come header, footer, article e section per migliorare la leggibilità e l'accessibilità.
Commenti nel codice: Usa i commenti per annotare il tuo codice:
html
Copia codice
!-- Questo è un commento --
Validazione: Utilizza strumenti di validazione per assicurarti che il tuo codice sia conforme agli standard HTML.

Conclusione Questo manuale ti ha fornito una panoramica delle basi dell’HTML. Con pratica e sperimentazione, potrai creare pagine web sempre più complesse e interattive. Buona fortuna nel tuo viaggio nel mondo della programmazione web!
